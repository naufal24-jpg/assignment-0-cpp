# REPORT - Programming Assignment (Hello World C++)

## 1. Deskripsi Program

Program ini dibuat untuk menerima input berupa nama dari pengguna, kemudian menampilkan output berupa sapaan (greeting) menggunakan nama tersebut.

Program ini merupakan pengembangan dari program "Hello World" dengan tambahan input dari user.

---

## 2. Input dan Output

### Input

Program menerima input berupa string (nama pengguna).

Contoh:

```id="a1b2c3"
Naufal
```

---

### Output

Program akan menampilkan sapaan dengan format:

```id="d4e5f6"
Hello Naufal
```

---

## 3. Algoritma Program

Langkah-langkah program:

1. Deklarasi variabel `nama` bertipe string
2. Menampilkan pesan untuk meminta input dari pengguna
3. Mengambil input menggunakan `getline()`
4. Menampilkan output berupa "Hello" diikuti nama yang diinputkan

---

## 4. Struktur Program

Program menggunakan:

* `#include <iostream>` untuk input/output
* `getline(cin, nama)` untuk membaca input satu baris penuh
* `cout` untuk menampilkan output

---

## 5. Kesimpulan

Program ini menunjukkan penggunaan dasar input dan output dalam C++. Dengan menggunakan `getline()`, program dapat menerima input berupa kalimat atau nama lengkap dari pengguna.
